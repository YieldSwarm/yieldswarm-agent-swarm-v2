# YieldSwarm Deployment

## Akash Deployment (Primary Path)

1. Install Akash CLI
2. Fund your wallet
3. Create deployment using `deploy-swarm-monolith.yaml`
4. Extract worker URLs
5. Wire into Arena frontend
